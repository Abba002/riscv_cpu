# RISC-V Processor Project

## Overview

This project implements a 32-bit RISC-V processor in Verilog. The goal is to understand processor architecture and digital design fundamentals by building the major components of a CPU from the ground up.

The processor will eventually include:

* Register File
* Arithmetic Logic Unit (ALU)
* Control Unit
* Instruction Memory
* Data Memory
* Single-Cycle Datapath
* Five-Stage Pipeline

## Project Goals

* Learn Verilog hardware design
* Understand processor architecture
* Practice verification and debugging
* Build a portfolio project for semiconductor industry roles
